# C 回调函数练习题
1. 最简单回调 —— 函数指针调用
标题意思：最基础的 “把函数当参数传”，理解什么是回调。
题目：写一个函数 run()，接收一个无参无返回函数，调用它。实现：run(hello) 输出 Hello callback!。
核心：函数指针、回调基本格式。

1. 带参数回调 —— 处理单个 int
标题意思：回调可以带参数，不是只能空参数。
题目：写 process(int val, void (*cb)(int))，把 val 传给回调处理。实现打印、平方等。
核心：带参回调。

1. 遍历数组通用框架（最经典）
标题意思：遍历逻辑固定，处理逻辑可变 → 用回调实现复用。
题目：写通用遍历 traverse(arr, len, cb)，回调分别实现：打印、平方、立方。
核心：代码复用、解耦。

1. 带返回值回调 —— 过滤 / 判断
标题意思：回调可以返回 int 表示真假，用来做条件筛选。
题目：写 filter(arr, len, pred)，pred 返回 1/0。筛选偶数、正数、>5 的数。
核心：判断型回调、过滤模式。

1. 回调带 “上下文”（解决全局变量脏问题）
标题意思：用 void* ctx 把主函数变量传给回调，不用全局变量。
题目：遍历数组求和、求最大值，结果存在主函数变量里。
核心：void* 上下文、回调传数据。

1. 排序通用库 —— qsort 风格（超级重要）
标题意思：模仿 C 标准库 qsort，排序框架固定，比较规则用回调。
题目：实现通用排序 mySort，回调决定升序 / 降序。
核心：通用算法 + 回调策略。

1. 字符串处理 —— 过滤长度
标题意思：回调不只处理 int，可处理任意类型（字符串）。
题目：筛选字符串长度 > 3、以 'a' 开头等。
核心：通用回调扩展到字符串。

1. 事件回调 —— 按钮点击模拟
标题意思：事件触发时调用回调，GUI / 嵌入式常用。
题目：定义 Button 结构体，包含 onClick 回调，模拟点击。
核心：事件驱动、结构体 + 回调。

1. 异步 / 延时回调（定时器思想）
标题意思：未来某个时刻才调用回调，模拟定时器 / 异步。
题目：setTimeout(ms, cb) 延时后执行 cb。
核心：延时 / 异步回调模型。

1.  多级回调 —— 遍历 + 过滤 + 处理
标题意思：两层回调组合：条件回调 + 处理回调，高度解耦。
题目：processIf(arr, len, cond, proc)满足 cond 才执行 proc。
你刚才写的 “先存立方再打印” 就是这题的工程升级版。
核心：组合回调、真正框架级架构。
