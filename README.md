# C 语言回调函数 10 道经典练习题（由浅入深）
专为 C 语言新手设计，从基础函数指针到工程级回调框架，循序渐进掌握回调核心思想。

## 1. 最简单回调 —— 函数指针调用
**核心**：函数指针、回调基本格式
**题目**：编写一个 `run()` 函数，接收一个无参无返回值的函数作为参数并调用它。实现 `run(hello)` 输出 `Hello callback!`。

## 2. 带参数回调 —— 处理单个 int
**核心**：带参回调
**题目**：编写 `process(int val, void (*cb)(int))` 函数，将 `val` 传递给回调函数处理。分别实现打印数值、计算平方的回调功能。

## 3. 遍历数组通用框架（最经典）
**核心**：代码复用、解耦
**题目**：编写通用数组遍历函数 `traverse(arr, len, cb)`，遍历逻辑固定，处理逻辑由回调决定。分别实现打印数组元素、计算平方、计算立方的回调。

## 4. 带返回值回调 —— 过滤 / 判断
**核心**：判断型回调、过滤模式
**题目**：编写 `filter(arr, len, pred)` 函数，回调函数 `pred` 返回 1/0 表示真假。实现筛选偶数、正数、大于 5 的数字等功能。

## 5. 回调带 “上下文”（解决全局变量问题）
**核心**：`void*` 上下文、回调传数据
**题目**：遍历数组求和、求最大值，不使用全局变量，通过 `void*` 上下文将主函数的变量传递给回调，结果直接存储在主函数变量中。

## 6. 排序通用库 —— qsort 风格（超级重要）
**核心**：通用算法 + 回调策略
**题目**：模仿 C 标准库 `qsort`，实现通用排序函数 `mySort`，排序规则（升序/降序）由回调函数决定。

## 7. 字符串处理 —— 过滤长度
**核心**：通用回调扩展到字符串类型
**题目**：将回调应用于字符串数组，筛选出长度大于 3、以字符 `'a'` 开头的字符串，掌握回调处理任意类型的能力。

## 8. 事件回调 —— 按钮点击模拟
**核心**：事件驱动、结构体 + 回调
**题目**：定义 `Button` 结构体，包含 `onClick` 回调函数指针，模拟按钮点击事件，触发时执行对应回调（嵌入式/GUI 常用）。

## 9. 异步 / 延时回调（定时器思想）
**核心**：延时 / 异步回调模型
**题目**：实现 `setTimeout(ms, cb)` 函数，模拟延时操作，等待指定毫秒后执行回调函数，理解异步回调思想。

## 10. 多级回调 —— 遍历 + 过滤 + 处理
**核心**：组合回调、框架级架构
**题目**：编写 `processIf(arr, len, cond, proc)` 函数，结合**条件回调**和**处理回调**，只有满足条件回调时，才执行处理回调，实现高度解耦的工程级逻辑。
